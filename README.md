# bing-skills

A collection of agent skills for Claude Code, Cursor, Codex, and 70+ other coding agents. Install any skill with one command.

## Install

```bash
npx skills add 1767249060-creator/bing-skills
```

Install a single skill:

```bash
npx skills add 1767249060-creator/bing-skills --skill agent-geo
```

Works with any agent supported by [`npx skills`](https://skills.sh) — Claude Code, Cursor, Codex, Copilot, Gemini CLI, and more.

## Skills

| Skill | What it does |
| --- | --- |
| [agent-geo](skills/agent-geo/) | GEO for your own repos, skills, and MCP servers — get them discovered, trusted, and invoked by other people's AI agents. Covers skills.sh ranking mechanics, description-as-routing-rule writing, monorepo strategy, cold-start, and a pre-publish checklist. |
